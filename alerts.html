Database Schema

The alerts table structure in motion_alerts.db

SQL
CREATE TABLE IF NOT EXISTS alerts (
    id          INTEGER PRIMARY KEY AUTOINCREMENT,
    timestamp   TEXT    NOT NULL,
    photo_path  TEXT,
    message     TEXT
);

How to View Your Alerts

🖥️

Terminal

Run python show_alerts.py to print all alerts in the terminal and auto-export to CSV.

📊

CSV File

Open alerts_export.csv in Excel or Google Sheets for easy viewing and sorting.

🗄️

DB Browser

Use DB Browser for SQLite (free app) to open motion_alerts.db and browse records visually.
