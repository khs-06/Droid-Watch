Setup Guide

Get PhoneGuard running in a few minutes

Requirements

🐍 Python 3.8+ Required to run all scripts
📱 Android Phone With IP Webcam app installed
📶 Same Wi-Fi Phone & PC on same network
🤖 Telegram Account To receive alerts

Installation

Step 1

Clone the Repository

bash 
git clone https://github.com/whitehatboy005/Motion-Detection-Alert-System-for-CCTV
cd Motion-Detection-Alert-System-for-CCTV
Step 2

Install Dependencies

bash 
pip install -r requirements.txt

This installs: opencv-python, requests, python-dotenv, PyAutoGUI, pillow, python-telegram-bot

Step 3

Get Your Telegram Chat ID

Open Telegram and message @GetMyChatID_Bot. It will reply with your Chat ID number. Save it — you'll need it in the next step.

Step 4

Find Your ROI Coordinates

bash 
python ROI_point_finder.py

A fullscreen overlay appears. Click and drag to draw your detection zone. The coordinates will print in the terminal.

Step 5

Edit config.env

bash 
notepad config.env
config.env 
CHAT_ID = your_chat_id_here
ROI_START_POINT = 100,100
ROI_END_POINT = 400,300
IP_CAMERA_URL = http://192.168.x.x:8080/video
Step 6

Start Telegram Bot

Open Telegram → Search @CAMSEC_AlertBot → Press START. This is required before running the script.

Step 7

Start IP Webcam on Your Phone

Open the IP Webcam app on your Android phone → scroll to bottom → tap "Start Server". Note the IP address shown on screen and use it as your IP_CAMERA_URL.

Step 8

Run the Main Program

bash 
python Phone_Alert.py
✅ A live camera window opens. Motion in your ROI will trigger a Telegram alert!
Optional

Verify Configuration

While the script is running, open Telegram and type /check to the bot. It will reply with your current settings.

Telegram 
/check
Optional

View Past Alerts

bash 
python show_alerts.py

Prints all logged alerts and exports them to alerts_export.csv
