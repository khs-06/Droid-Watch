Features

Everything PhoneGuard can do for you

📱

Phone Camera as CCTV

Install the free IP Webcam app on your Android phone and it becomes a wireless camera. PhoneGuard connects to it over your Wi-Fi network — no cables, no extra hardware needed.

IP Webcam App Wi-Fi Streaming
🎯

Region of Interest (ROI) Selection

Run ROI_point_finder.py and a fullscreen overlay appears. Click and drag to draw a box around exactly the area you want to monitor. Only motion inside that box triggers alerts.

Visual GUI Tool Zero False Alerts
🔍

Background Subtraction Detection

Uses OpenCV's MOG2 background subtractor to detect moving objects. Only triggers when an object is detected for multiple consecutive frames — avoiding false alerts from lighting changes.

OpenCV MOG2 Persistence Threshold 10s Cooldown
🚨

Instant Telegram Alerts with Snapshot

The moment motion is confirmed, PhoneGuard captures a photo of the frame and sends it to your Telegram chat with the caption "🚨 Motion Detected! 📱 Phone Camera Alert".

Photo Snapshot Telegram Bot
⚠️

Camera Disconnect Alert

If the phone camera stream is lost or disconnects, PhoneGuard immediately sends a Telegram text message warning you to check your phone.

Connection Monitoring Auto Retry
🗄️

SQLite Database Logging

Every alert is automatically saved to motion_alerts.db with the timestamp, photo path, and message. The database is created automatically — no setup needed.

Auto-created DB Timestamp Logging
🤖

Telegram Bot Command: /check

Type /check in Telegram and the bot replies with your current configuration — camera URL, ROI coordinates, and Chat ID. Quick way to verify everything is set up correctly.

/check Command Live Config Status
📊

CSV Export of All Alerts

Run show_alerts.py to print all past alerts from the database and automatically export them to alerts_export.csv for review or analysis.

CSV Export Alert History
