Step by Step

01

Install & Configure

Clone the repo and install all Python dependencies. Then open config.env and fill in:

  • CHAT_ID — Get this from @GetMyChatID_Bot on Telegram
  • IP_CAMERA_URL — The stream URL from IP Webcam app (e.g. http://192.168.x.x:8080/video)
  • ROI_START_POINT — Top-left corner of your detection zone
  • ROI_END_POINT — Bottom-right corner of your detection zone
02

Find Your ROI Coordinates

Run python ROI_point_finder.py. A fullscreen screenshot appears as an overlay. Click and drag to draw a green rectangle around the area you want to monitor. The start and end coordinates are printed in the terminal — copy them into your config.env file.

03

Start Telegram Bot

Open Telegram and search for @CAMSEC_AlertBot. Press START. This allows the bot to send messages to your chat. If you don't do this step, alerts won't be delivered.

04

Open IP Webcam on Your Phone

Install the IP Webcam app on your Android phone. Open it and tap "Start Server". Your phone will show an IP address and port — use this as your IP_CAMERA_URL. Make sure your phone and PC are on the same Wi-Fi network.

05

Run the Main Script

Run python Phone_Alert.py. A window opens showing your phone's live camera feed with a green ROI box drawn on it. The system is now actively monitoring.

💡 The script runs two things at once: the motion detection loop (in a background thread) and the Telegram bot listener (main thread).
06

Motion Detected → Alert Sent

When the system detects persistent motion inside the ROI for 5+ consecutive frames, it:

  • Saves a JPEG snapshot as motion_alert.jpg
  • Sends it to your Telegram chat with a caption
  • Logs the event to motion_alerts.db with a timestamp
  • Waits 10 seconds before triggering another alert (cooldown)
07

Verify Config with /check

At any time, type /check in the Telegram bot chat. It replies with your current camera URL, ROI coordinates, and Chat ID so you can confirm everything is configured correctly.

How Motion Detection Works

Read Frame
Captures each frame from the camera stream using OpenCV

↓
✂️

Crop ROI

↓
🔲

Background Subtraction
MOG2 algorithm compares current frame to the learned background

↓
🔍

Find Contours
Looks for moving blobs bigger than 5000px² (filters out noise)

↓
🚨

Trigger Alert
If motion persists for 5+ frames and cooldown has passed → send alert
